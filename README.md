# 历代状元籍贯分布

> 将中国历代科举状元（隋朝 605 年 → 清朝 1904 年）的籍贯按现代省份汇总，在交互式地图上直观呈现。

🔗 **在线预览**：[https://sanmingmrx.github.io/ZhuangyuanRank/](https://sanmingmrx.github.io/ZhuangyuanRank/)

---

## 功能

- 🗺️ **中国地图热力展示** — 省份颜色深浅反映状元人数多少
- 🔢 **省份排行榜** — 实时显示各省状元总数及主要朝代
- 🏛️ **朝代多选筛选** — 隋 / 唐 / 五代 / 北宋 / 南宋 / 辽 / 金 / 元 / 明 / 清
- 🔗 **地图 ↔ 列表联动** — 点击地图高亮列表，点击列表高亮地图
- 📜 **状元详情展开** — 点击省份查看该省历代状元姓名与年份

## 数据说明

- 收录状元约 **474 位**，覆盖 **18 个省份**
- 数据来源：《登科记考》《历代状元传》《明清历科进士题名碑录》等史料
- 虚线标注的条目为籍贯或身份存在史料争议的记录
- 籍贯均按今日省级行政区标注

## 使用方式

直接用浏览器打开 `index.html`（需联网加载地图数据），或访问上方在线预览链接。
